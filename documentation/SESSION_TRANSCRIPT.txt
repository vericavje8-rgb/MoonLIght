FAZA 8 — MOBILE NAV FIX
--------------------------------------------------------------------------------

[Alvin] "menu item in menu on mobile is dark"
[Slika] Screenshot pokazuje tamne (crne) nav linkove na mobileu.

[Razmišljanje] Header je na mobileu dobio background: var(--primary) (tamni),
               ali nav a nasljeđuje body color (#2d1810 — tamno smeđe/crno)
               umjesto --white. Bazni nav a { color: var(--white) } vrijedi
               samo kad nema konfliktnog nasljeđivanja, ali body { color: var(--text-dark) }
               i a { color: inherit } zajedno override-aju to na mobileu.

[Akcija] Dodan explicit override u @media (max-width: 768px):
         nav a { color: #ffffff; }
         Commit: 8da2781 — "Fix nav: always solid background on mobile"

[Alvin] "its black, letters" — problem i dalje postoji.

[Razmišljanje] color: var(--white) u nav a definiciji nije dovoljan jer
               a { color: inherit } dolazi ispred i nasljeđuje body text-dark.
               Eksplicitni #ffffff u media query-u treba nadpisati sve.

[Akcija] Provjera — nav a u media query-u možda nije dovoljno specifičan.
         Dodan još jedan explicit: nav a { color: #ffffff; } unutar @media 768px bloka.
         Commit: 73c787a — "Fix nav link color on mobile (force white)"
